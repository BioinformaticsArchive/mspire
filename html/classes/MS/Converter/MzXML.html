file_to_mzxml(file)

first, converts backslash to forward slash in filename. if .mzXML returns the filename if .raw or .RAW converts the file to .mZXML and returns mzXML filename if no recognized extension, looks for .mzXML file, then .RAW file (and converts) aborts if file was not able to be converted returns nil if a file that can be converted or used was not found
