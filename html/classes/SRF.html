Public Class methods

new(filename=nil, peps=[], global_ref_hash={})

peps and global_ref_hash are created as the srf files is read. If the file is read as part of a group, then these should be passed in. NOTE: if you want the hits filtered by precursor tolerance (the way they might be displayed in .out files) you should probably use SRFGroup (which does this by default) SRF is meant to be a low level read of the file.
